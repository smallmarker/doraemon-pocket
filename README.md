# Doraemon Pocket

## About

Doraemon Pocket is a lightweight, open-source directory for useful online tools.

It is designed for people who know what they need to do but do not remember which website can help. Try searching for the tool you need, and Doraemon Pocket will point you to a curated external tool.

The project is intentionally simple: it does not perform tasks, generate content, collect user data, or run a backend service. It is a static, browser-based search experience built around a maintained collection of tool links.
